# Dashboard Overview

The TurbineProxy dashboard provides real-time visibility into your proxy and database backends. See [Dashboard](../features/dashboard) for the complete feature documentation.

## Quick Navigation

| Panel | What you'll find |
|---|---|
| Overview | Global counters, read/write ratio |
| Queries | Top queries by count and P95 latency |
| Heatmap | Latency distribution over time |
| Throughput | Queries per minute/hour/day |
| N+1 | Repeated query pattern detection |
| Pool | Connection pool utilization |
| Backends | Backend health and replication lag |
| Cluster | Group Replication / HA cluster state |
| Errors | Last 1000 backend errors |
| Config | Runtime rule management |
| Transactions | Active long-running transactions |
| Users | Per-user connection counts |
